WEBSITE

Defination

A website is a series of webpages bound together by a common domain name and displayed on web servers. Essentially, a website is a home for all the information you want to put out on the web. The information on your site is typically made up of a combination of text, images, videos and icons that come together to tell a unified story.

WEBSITE COMPONENTS

  1. Web Hosting

    2. Web host gives a home to websites and enables them to be displayed online. Web hosts are servers that store the information that stands behind each and every website. These servers work to ensure that your site will appear on a visitor's screen whenever they try to access it.

  2. Domain Name

    3. A domain name is the web address of your site. This is what makes it easily identifiable and findable online.Domain names have to be purchased through official domain registrars as no two domain names can be exactly the same.Often, domain names are the same as a brand name. For example, if you have a brand name olive, you might have a domain name of olive.net.

TYPES OF WEBSITE

Following are the types of Website

  1. Static Website

    2. Static websites are sites that do not adapt, but instead have fixed elements. They are made up of pages coded with HTML and always appear the same unless you go into the backend of a page to change its code. These are often the simplest form of website to create as they do not require any form of flexibility. Any website type can be static, it simply requires greater upkeep should you choose to make a change to your site.

  2. Dynamic Website

Dynamic websites are sites that adapt according to circumstances and users. They utilize databases which can then present varying information across many pages depending on a number of factors. These might include who is accessing the pages, when they are being accessed, and more. This strategy is often used by large websites since the maintenance is lower. For example an E-Commerce website with a shopping cart would have to be dynamic as it changes per user.

LANGUAGES USED FOR MAKING WEBSITES

  1. HTML

    2. HTML stands for Hypertext Markup Language. It is Front end language used to structure a web page and its content. For example, content could be structured within a set of paragraphs, a list of bulleted points, or using images and data tables.HTML is used to define the content of web page

  2. CSS

    3. CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once.External stylesheets are stored in CSS files. There are 3 types of CSS.

    1. Internal CSS
    2. External CSS
    3. Inline CSS

  3. BOOTSTRAP

    4. Bootstrap is a potent front-end framework used to create modern websites and web apps. It's open-source and free to use, yet features numerous HTML and CSS templates for UI interface elements such as buttons and forms. Bootstrap also supports JavaScript extensions.

  4. JAVA SCRPT

    5. JavaScript is the programming language of the Web.JavaScript is easy to learn.Java Script is used to express the behaiviour of web page. Over 97% of website use java script on the client side for web page behaviour.

  5. PHP

PHP stands for Hypertext PreProcessor. It is an open source general purpose server scripting language, and a powerful tool for making dynamic and interactive Web pages.PHP is a widely-used, free, and efficient alternative to competitors such as Microsoft's ASP.It can be embedded into HTML.
